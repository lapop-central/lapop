# LAPOP Fonts (design)

This function loads fonts needed for LAPOP graph formatting. In contrast
to lapop_fonts(), this renders text as text instead of polygons, which
allows post-hoc editing.

## Usage

``` r
lapop_fonts_design()
```

## Value

No return value, called for side effects

# LAPOP Fonts

This function loads fonts needed for LAPOP graph formatting. No
arguments needed; just run lapop_fonts() at the beginning of your
session.

## Usage

``` r
lapop_fonts()
```

## Value

No return value, called for side effects

## Author

Luke Plutowski, <luke.plutowski@vanderbilt.edu> & Robert Vidigal,

## Examples

``` r
require(lapop); lapop_fonts()
#> LAPOP fonts loaded successfully: Inter, Roboto, and Nunito (regular and light).
```

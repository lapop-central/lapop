# bra23: Single-country Single-year Dataset

A dataset containing the World Map geometry for lapop_map() function.

## Usage

``` r
world
```

## Format

A data frame

- pais:

  Country name

- pais_lab:

  Country ISO2 code

- geometry:

  Polygon with geometry

## Source

LAPOP AmericasBarometer (https://www.vanderbilt.edu/lapop/)
